# 💰 Personal Finance Intelligence System

An interactive **AI powered financial analytics dashboard** built using Streamlit that helps users understand spending behavior, detect anomalies, and forecast future expenses.



## 📌 Overview

This project transforms raw transaction data into actionable financial insights.

---

## 🎯 Features

### 📊 Expense Analysis
- Category-wise spending breakdown  
- Interactive visualizations using Plotly  

### 📈 Trend Analysis
- Monthly expense tracking  
- Behavioral pattern detection  

### 🚨 Anomaly Detection
- Identifies unusual spending patterns  
- Alerts users for financial irregularities  

### 🤖 AI Prediction
- Forecasts next month’s expenses  
- Uses Linear Regression  

### 💡 Smart Insights
- Highlights top spending categories  
- Provides financial recommendations  
- Detects overspending vs savings  

### 🌐 Interactive UI
- Landing page (website-style interface)  
- Dashboard layout  
- Upload your own dataset
